Today in Christory, in 2008, Chris' PS3 History level is given credit on the LBP site, gladdening Chris, but enraging many of the game's fanboys.
(more...)
Usage
These usage notes will not be included in articles bearing this template.This template sits pride of place on the CWCki main page to provide readers with an insight in to what happened in Chris's life on that day. The template automatically displays today's entry, and will provide a default message if the day does not exist yet. Do not attempt to edit this template to adjust the current Christory entry; instead, find the relevant day template from the list below.
Christory is a work in progress (part of CWChronology) so feel free to add to the templates, telling us what other significant events happened on that day. Each template should use {{Christory Entry}} (Usage), instructions on how to use it can be found on the template page.
